# Fresh-Mart Support Portal

This repository contains the source code and documentation for the Fresh-Mart Support Portal, a centralized knowledge base designed to provide operational standards and instructional guides for staff members.

## Project Overview

Fresh-Mart is a retail simulation environment focused on balancing a professional work atmosphere with an engaging player experience. This website serves as the primary resource for:

* **Operational Standards:** Guidelines on store maintenance and customer interaction.
* **Rank-Specific Information:** Localized instructions for Low Rank (LR), Middle Rank (MR), and High Rank (HR) staff members.
* **Policy Documentation:** Clear outlines of community rules and punishment guides.

The site is built using Docusaurus, a modern static site generator, to ensure fast loading times and a structured navigation experience without file extensions in the URL.

## Contributing

Contributions to improve the accuracy and clarity of the documentation are welcome. To contribute, follow these steps:

1. **Report Inaccuracies:** If you find outdated information, contact the maintainer or open an issue in this repository.
2. **Submit Changes:**
    * Fork the repository.
    * Create a new branch for your updates.
    * Ensure all markdown files follow the established formatting.
    * Submit a pull request with a detailed description of the changes.
3. **Style Guidelines:**
    * Keep language professional and concise.
    * Ensure all links are functional before submitting.
    * Stick to factual group policies.
